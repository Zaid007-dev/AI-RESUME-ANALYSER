## 🚀 About The Project

**AI Resume Analyzer** is a smart web application that evaluates resumes using advanced AI models from **OpenAI** and **Google Gemini**.

It helps job seekers improve their resumes and enables recruiters to quickly analyze candidate profiles with intelligent feedback.

📄 Upload your resume → 🧠 AI analyzes → 📊 Get actionable insights instantly.

💡 Whether you're a **student**, **job seeker**, or **recruiter**, this tool helps you make data-driven career decisions.

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 🔧 1. Clone the Repository

```bash
git clone https://github.com/your-username/ai-resume-analyzer.git
cd ai-resume-analyzer
```

### 📦 2. Install Dependencies

```bash
pip install streamlit pdfplumber python-docx google-generativeai openai python-dotenv
```

### 🔑 3. Setup Environment Variables

Create a `.env` file in the root directory and add:

```env
OPENAI_API_KEY=your_openai_api_key
GOOGLE_API_KEY=your_gemini_api_key
```

### ▶️ 4. Run the Application

```bash
streamlit run app.py
```

## 🧠 How It Works

1. 📤 Upload your resume (PDF/DOCX)
2. 🔍 Text is extracted using parsing libraries
3. 🤖 AI models (OpenAI + Gemini) analyze content
4. 📊 System generates insights like:
   - Skills match
   - Resume score
   - Improvement suggestions
5. 📈 User gets a detailed report instantly
